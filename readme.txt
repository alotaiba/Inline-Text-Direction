Changes the direction of inline text that is written as "RTL" that are embedded within Latin languages that are written as "RTL". and vice versa.

== Description ==

Inline Text Direction plugin for WordPress changes the direction of inline foreign text that is written from "Right to Left" that are embedded within Latin languages that are written from "Left to Right". and vice versa.

In general, when foreign text that has a different directional run than the paragraph that is embedded within, it’ll look very badly, that even sometimes, it’s hard to understand the paragraph due to this mix.

&#x202B;تقوم إضافة "اتجاه النص المضمن" لنظام WordPress بتغيير اتجاه النصوص الأجنبية التي تُكتب من "اليسار إلى اليمين" داخل الفقرات العربية والتي تُكتب من "اليمين إلى اليسار"، أو العكس.&#x202C;

&#x202B;في العادة، وعند دمج الكلمات الأجنبية في الفقرات العربية تظهر بشكل سيء، ويتغير اتجاه النصوص، بحيث في بعض الأحيان يصعب فهم المعنى لتداخل هذه النصوص ببعضها البعض.&#x202C;

== Installation ==

To install this plugin, you need to do the following:

1. Unzip the downloaded file from the WordPress plugins directory.
1. Upload the unzipped "**inline-text-direction**" directory to your WordPress plugin directory "wp-content/plugins".
1. Activate the plugin.
1. Optional: You’ll find under "Options" a new link to control this plugin.

1. &#x202B;رفع المجلد "**inline-text-direction**" إلى الخادم في مسار إضافات WordPress وهو "/wp-content/plugins"&#x202C;
1. &#x202B;تفعيل الإضافة من لوحة التحكم.&#x202C;

== Usage ==

In the Visual Editor (tinyMCE), You can select the foreign inline text you want to chnage the direction for, then, you can click the button "**The direction of foreign inline text**" which you can find on the toolbar.

In the HTML editor, you can also change the direction of inline text, and you can do by selecting the desired text, and then click on "**><**" which you can find on the toolbar.

&#x202B;في محرر النصوص المتطور tinyMCE، يمكنك تحديد النص الأجنبي المراد تحويل اتجاهه، ومن ثمّ الضغط على زر "**تغيير توجيه النص المضمن**"، والذي ستجده من بين الأزرار في أعلى المحرر.&#x202C;

&#x202B;في محرر HTML، يمكنك أيضاً تحديد النص الأجنبي المراد تحويل اتجاهه، ومن ثمّ الضغط على على "**><**"، والذي ستجده من بين الأزرار في أعلى المحرر.&#x202C;
